## About

**Brubru Nails** is a digital product brand created by Bruna, a nail designer with over 10 years of experience. The brand offers digital planners, courses, and tools to help nail professionals organize and grow their businesses.

This repository contains the official website built with **Astro** and **Tailwind CSS**.

## Features

- Responsive landing page with GSAP animations
- Product catalog with digital planners and courses
- Blog section for content marketing
- Contact form and newsletter integration
- SEO optimized with structured data
- GitHub Actions CI/CD for automatic deployment

## Tech Stack

| Technology | Purpose |
|------------|---------|
| [Astro](https://astro.build) | Static site generator |
| [Tailwind CSS](https://tailwindcss.com) | Utility-first CSS framework |
| [TypeScript](https://typescriptlang.org) | Type-safe JavaScript |
| [GSAP](https://greensock.com/gsap) | Professional-grade animations |
| GitHub Pages | Hosting & deployment |

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
brubru-nails/
```
